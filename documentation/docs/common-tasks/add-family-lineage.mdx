# Add a family lineage

## Goal

Create a family/lineage entity and associate people with it and with known relatives.

## Steps

1. Open **Taxonomies & Entities → Families / Lineages** and create a new document.
2. Enter the **Family Name** (e.g. The Bean Family). Optionally add **Historical Background**.
3. Open each relevant **Historical Person** under **Taxonomies & Entities → Historical Persons**.
4. Under **Family Lineages**, add a reference to the lineage document.
5. Under **Known Immediate Relatives**, add entries for specific relationships: choose the relative’s person profile and a relationship type (Spouse, Parent, Child, Sibling, Cousin, or Other).
6. Publish the person documents (and the lineage) when ready.

## Field notes

- Prefer one lineage document per family group; reuse existing lineages instead of near-duplicates.
- Relationships are directional from this person’s perspective (e.g. “Parent” means the linked relative is a parent of this person).
- Prefer linking existing person documents over creating duplicates; use aliases on the person when spellings vary.
