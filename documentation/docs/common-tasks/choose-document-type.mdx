# Choose the right document type

## Goal

Create the correct document type so archive transcriptions, website essays, and Quarterly citations stay distinct.

## Decision guide

| You are working with… | Create this type | Desk section |
| --- | --- | --- |
| One historical clipping, letter, or notice to transcribe | **Primary Source** | The Archive |
| A photograph to catalog | **Historical Image** | The Archive |
| Modern long-form research or an interactive overview page | **Curated Essay** | The Website |
| A digitized TEHS Quarterly article (full text, page breaks, entity tags) | **TEHS Quarterly Article** | The Website |
| A person, family lineage, place, property, business, or subject theme | Matching entity under **Taxonomies & Entities** | Taxonomies & Entities |

## Field notes

- Primary Source schema description: use it **only** for a single historical piece of media. Essays and modern research belong in Curated Essay.
- Quarterly Article documents store the **full article text** plus publication metadata and entity links. Archive items can still reference them under Research References.
