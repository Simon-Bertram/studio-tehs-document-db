# Model a property or estate

## Goal

Describe a building or land tract, including parent estates, residents, and deed references.

## Steps

1. Open **Taxonomies & Entities → Properties & Buildings** and create a new document.
2. Set **Historical Name** (required) and **Property Type** (estate, dwelling, outbuilding, or subdivision lot).
3. If this is a house or outbuilding on a larger farm, set **Parent Estate / Land Tract** (hidden when type is Estate).
4. Set **Specific Location** or **Township** ([geography rules](./link-geography.mdx)).
5. Add structural evolution notes, notable residents, modern address, year built, and Chester County deed citations (e.g. `U15 P466`).
6. Link the property from primary sources via **Associated Property** when cataloging related notices.

## Field notes

- Example parent link: “By-The-Creek” → parent “Glenhardie Farm”.
- Deed citations are free-text strings matching familiar Chester County book/page forms.
