# Tag with subject categories

## Goal

Use subject categories so related primary sources and images can be discovered by theme.

## Steps

1. Ensure the category exists under **Taxonomies & Entities → Subject Categories** (create if needed: title + optional description).
2. Open a **Primary Source** or **Historical Image**.
3. In **Research → Subjects**, add references to the relevant categories (e.g. Schools, Railroads, Farms).
4. Publish when ready.

## Field notes

- Categories are broad themes from subject lists and advanced-search style filters—not free-text tags on every document.
- Prefer reusing existing category documents over creating near-duplicates.
