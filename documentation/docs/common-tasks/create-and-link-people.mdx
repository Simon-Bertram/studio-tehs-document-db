# Create and link people

## Goal

Maintain historical person records and reference them from sources, properties, and businesses.

## Steps

1. Open **Taxonomies & Entities → Historical Persons** and create a new document (or find an existing one).
2. Enter first and last name (required). Add prefix/suffix when useful (Capt., Jr., etc.).
3. Optionally set born and died dates (died must be on or after born).
4. Add **Alternate Spellings / Aliases** for variants found in records.
5. Add **Census / Occupation Records** (year + occupation) when known.
6. From a primary source, property, or business, link the person under People Mentioned, Notable Residents, or Owners.

## Field notes

- Prefer one person document per individual; use aliases instead of duplicate profiles.
- Sources use **People Mentioned**; properties use **Notable Residents / Owners**; businesses use **Owners / Operators**.
