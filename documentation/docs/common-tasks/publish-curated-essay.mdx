# Publish a curated essay

## Goal

Create a long-form website page with slug, township scope, and optional map or essay-link modules.

## Steps

1. Open **The Website → Curated Essays & Overviews** and create a new document.
2. Set **Page Title** and generate the **URL Slug** (both required).
3. Add **Townships** that define the essay’s geographic scope.
4. In **Page Content & Layout Canvas**, write Portable Text content and insert images as needed.
5. Optionally insert an **Interactive Map Module** with historical map year and engine embed URL.
6. Optionally insert a **Nested Navigation Portal Index** linking to other curated essays.
7. Publish when ready.

## Field notes

- Use Curated Essay for modern research and overviews—not for raw clipping transcriptions ([choose the right type](./choose-document-type.mdx)).
- Map modules need both a target year and an embeddable application URL for the map engine.
