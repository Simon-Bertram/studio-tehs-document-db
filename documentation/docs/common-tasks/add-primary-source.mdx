# Add a primary source / transcription

## Goal

Ingest a single historical clipping or letter with scan, transcription, dates, and links.

## Steps

1. Open **The Archive → Primary Sources / Transcriptions** and create a new document.
2. In **Identity**, set the headline/title (required). Add textual date (e.g. “circa 1890”) and/or exact publication date when known. Record the source publication name.
3. If matching a legacy record, set **Legacy Document ID** (e.g. `Doc505`)—it must be unique.
4. In **Content**, upload the scan and paste the full transcription.
5. In **Place**, prefer **Associated Property** when the place is known; otherwise set **Township**. Link people mentioned.
6. In **Research**, add subject categories and Quarterly citations as needed.
7. Publish when ready.

## Field notes

- Exact date helps sorting and filtering; textual date covers uncertain phrasing.
- Township is hidden once an associated property is set—put geography on the property instead.
- For sheriff’s sales, see [Enter a sheriff’s sale source](./enter-sheriff-sale.mdx).
