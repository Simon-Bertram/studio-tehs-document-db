# Enter a sheriff’s sale source

## Goal

Catalog a sheriff’s sale notice with legal writ type and, when known, the associated property.

## Steps

1. Create or open a **Primary Source** under **The Archive**.
2. Complete identity and content fields as for any transcription ([add a primary source](./add-primary-source.mdx)).
3. Enable **Is this a Sheriff's Sale?**
4. Set **Legal Writ Type** (Fieri Facias, Levari Facias, or Venditioni Exponas)—the field appears only when the flag is on.
5. Under **Place**, link **Associated Property** when the sale concerns a known property.
6. Publish when ready.

## Field notes

- Legal writ is hidden until the sheriff’s sale flag is enabled.
- Prefer property links over a bare township for sales tied to a specific tract.
