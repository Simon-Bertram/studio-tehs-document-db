# Cite a TEHS Quarterly article

## Goal

Maintain Quarterly bibliographic records and attach them as research references on archive items.

## Steps

1. Open **The Website → TEHS Quarterly Articles** and create a new document (or find an existing one).
2. Set article title (required). Add author, volume, issue, and published date.
3. Open the primary source or historical image that cites this article.
4. In **Research → Research References**, add a reference to the Quarterly article.
5. Publish both documents as needed.

## Field notes

- Quarterly documents store **citation metadata only**, not the full article text.
- Multiple archive items can cite the same Quarterly article.
