# Cite a TEHS Quarterly article

## Goal

Digitize a TEHS Quarterly article, link the historic sites and people it mentions, and attach it as a research reference on archive items.

## Steps

### Digitize the article

1. Open **The Website → TEHS Quarterly Articles** and create a new document (or find an existing one).
2. In **Publication Details**, set article title (required). Add author name, volume, issue, publication date (text), and start page.
3. In **Article Content**, transcribe the full article text into the body field.
4. Insert **Original Print Page Break** blocks where the original print pagination changes (e.g., at "Page 3") so academic citations remain accurate.
5. In **Tagged Entities**, link properties and people mentioned in the article (e.g., the historic sites listed at the end of a piece).

### Link from archive items

6. Open the primary source or historical image that cites this article.
7. In **Research → Research References**, add a reference to the Quarterly article.
8. Publish both documents as needed.

## Field notes

- The body field stores the full article text as Portable Text, not just citation metadata.
- Use page break blocks to preserve original print page numbers for citation without breaking the reading flow.
- Link properties and people in the entity arrays so the website can surface "Further Reading" connections across the archive.
- Multiple archive items can cite the same Quarterly article.
