# Catalog a historical image

## Goal

Add a photograph with caption, identifier, place, and provenance/rights metadata.

## Steps

1. Open **The Archive → Historical Images** and create a new document.
2. In **Identity**, set caption/title (required). Add identifier (e.g. `MF37`), serial number, and date taken as needed.
3. In **Content**, upload the photograph (required) and write a full description.
4. In **Place**, prefer **Specific Location**; use **Township** only when no more specific place exists.
5. In **Provenance / Rights**, record source, contributor, donation, photographer, rights, and archivist notes.
6. In **Research**, attach subjects and Quarterly citations.
7. Publish when ready.

## Field notes

- Identifier must be unique across historical images.
- Township field hides when a specific location is set (township comes from that location).
