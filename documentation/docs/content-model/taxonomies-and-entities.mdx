# Taxonomies & Entities

Shared records that organize and connect archive and website content. Create these once and reference them from many documents.

## Townships

Municipality-level geography (e.g. Tredyffrin, Easttown). Includes optional unique **legacy keyword** for old MySQL mapping.

## Locations

Villages or named places (e.g. Paoli, Berwyn, Cedar Hollow). Each location **must** belong to a parent township.

## Historical Persons

People mentioned in sources or tied to properties and businesses: name parts, born/died, alternate spellings, census/occupation records.

## Properties & Buildings

Estates, dwellings, outbuildings, and lots: historical name, type, optional parent estate, location/township, residents, modern address, year built, Chester County deed citations.

## Businesses

Named historical businesses with owner (person) and place (location or property) references.

## Subject Categories

Broad themes for discovery (e.g. Schools, Railroads, Farms). Tag primary sources and historical images so related material can be filtered.
